Contributing

Quetoo is open source and we welcome contributions of all kinds:

  • Code — Engine improvements, bug fixes, new features
  • Maps — New deathmatch levels and game content
  • Art — Textures, models, and visual effects
  • Testing — Play the game, report bugs, give feedback

Check out the GitHub repository to get started, or drop by Discord to say hello.
